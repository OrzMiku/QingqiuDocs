## 在线安装

如果您使用的启动器支持在线从 Modrinth 安装整合包，您可以直接搜索 `LunarFox` 安装整合包。

## 手动安装

如果您的启动器不支持在线安装，您可以手动下载整合包并导入启动器。

| 平台                                             | 下载链接                                                |
| ------------------------------------------------ | ------------------------------------------------------- |
| Modrinth <Badge text="推荐" variant="success" /> | [Modrinth](https://modrinth.com/modpack/lunarfox)       |
| Github                                           | [Release](https://github.com/OrzMiku/lunarfox/releases) |
| Github                                           | [Source](https://github.com/OrzMiku/lunarfox)           |

<Aside type="tip" title="最新构建">
  如果您希望跟随最新的整合包开发进度，您可以在 [Github
  Action](https://github.com/OrzMiku/lunarfox/actions)
  中获取到最新构建。您也可以在 Github上下载源代码，使用
  [packwiz](https://packwiz.infra.link/) 导出整合包。
</Aside>

## Pojav/FCL 注意事项

<Aside type="danger" title="兼容性警告">
  请注意，LunarFox 整合包并未针对移动端设计，因此不保证在 Pojav/FCL
  上的兼容性。在 Pojav/FCL（或其他移动端启动器） 上使用 LunarFox
  整合包可能会导致游戏崩溃或其他问题。请不要汇报 Pojav/FCL 上的任何问题。
</Aside>

如果您在 Pojav/FCL 上使用 LunarFox 整合包，下面是一些可能有效的建议：

- 禁用 ModernUI 模组，绝大多数情况下 ModernUI 模组会导致游戏崩溃。
- 使用支持 Sodium 的渲染器，如 Zink、LTW 渲染。
- 截止目前，LTW 渲染器暂不兼容 Xaero's WorldMap/MiniMap 模组，如有需要请使用其他地图模组。
- 截止目前，LTW 渲染器暂不（完全）支持着色器渲染。
