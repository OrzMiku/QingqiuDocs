<Aside type="note">
  请注意，LunarFox 整合包采用 Modrinth 格式进行发布，安装 LunarFox
  整合包需要您的启动器支持 Modrinth 打包格式。
</Aside>

## 在线安装

如果您使用的启动器支持在线从 Modrinth 安装整合包，您可以直接搜索 `LunarFox` 安装整合包。

## 手动安装

如果您的启动器不支持在线安装，您可以手动下载整合包并导入启动器。

| 平台                                             | 下载链接                                                |
| ------------------------------------------------ | ------------------------------------------------------- |
| Modrinth <Badge text="推荐" variant="success" /> | [Modrinth](https://modrinth.com/modpack/lunarfox)       |
| Github                                           | [Release](https://github.com/OrzMiku/lunarfox/releases) |
| Github                                           | [Source](https://github.com/OrzMiku/lunarfox)           |

<Aside type="tip" title="从源码导出">
  如果您希望跟随最新的整合包开发进度，您可以在 Github上下载源代码，使用
  [packwiz](https://packwiz.infra.link/) 导出整合包。
</Aside>
